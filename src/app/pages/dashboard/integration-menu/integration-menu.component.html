INTEGRASI API

Integrasi API adalah laman yang mendefinisikan secara lengkap data payload request dan response API oleh sistem yang mengakses IPortal Jamaah secara H2H (Host to Host).

Setiap request ke API akan membutuhkan token otentikasi. User yang digunakan untuk membuat token otentikasi dalam dibuat di menu user oleh Admin
